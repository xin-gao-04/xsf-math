# xsf-math / xsf-behavior 文档索引

本文档目录按两个逻辑层次组织：

- `xsf-math`
  算法层，负责数学模型、物理计算和领域公式。
- `xsf-behavior`
  行为层，负责把算法层组织成可由外部仿真框架托管的原子动作控制器。

## 总览文档

- `架构与构建分析.md`
  当前仓库的定位、构建方式、目录结构和双层模块关系。
- `集成指南.md`
  算法层和行为层的接入方式，以及最小使用面。

## 阅读顺序

1. 先看 `集成指南.md`，确认算法层和行为层的接入方式。
2. 再看 `架构与构建分析.md`，建立仓库分层和目录认知。
3. 需要公式与模型时，进入 `算法层/README.md`、`算法层/算法链路与逻辑详解.md` 和各业务域目录。
4. 需要理解当前飞行行为层时，先看 `行为层/飞行/飞行器飞行心智模型.md` 建立思维框架，再进入 `行为层/飞行/飞行行为工作流.md`。
