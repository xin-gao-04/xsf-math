# 基础知识整理

本文件对应电子战业务域，目的是在阅读 `electronic_warfare.hpp` 之前，先把干扰、诱饵和回波退化的基本思路整理清楚。

## 1. 这一层解决什么问题

电子战层关心的是：

- 干扰如何降低有效 SNR
- 箔条、诱饵和假目标如何改变感知链判决
- 雷达为什么在某些距离上更容易被压制，某些距离上更容易形成烧穿

它本质上是对感知链分母项和虚假目标项的建模。

## 2. 阅读代码前应先掌握的概念

### 2.1 J/S 和烧穿距离

雷达对抗里常见的不是单纯的 SNR，而是 J/S：

- `J`
  干扰功率
- `S`
  有效目标回波

随着距离变化，这两者的距离律不同，因此会出现烧穿距离。

### 2.2 自卫式干扰和站外干扰

- 自卫式干扰
  干扰机和目标共平台，主要影响目标自身生存。
- 站外干扰
  干扰机与目标分离，更适合区域掩护和支援压制。

### 2.3 箔条和假目标

电子战不一定只是在分母里加噪声。箔条和假目标还会改变雷达对场景的解释方式，使其产生错误跟踪或错误判决。

## 3. 与代码的对应关系

- `include/xsf_math/ew/electronic_warfare.hpp`
  提供干信比、烧穿距离、SNR 退化、箔条和假目标相关计算。

## 4. 常见误区

### 4.1 把电子战理解为“单纯增大噪声”

很多电子战手段更接近“改变雷达所看到的场景”，而不只是把噪声底抬高。

### 4.2 只用单一距离律估计 J/S

目标回波和干扰回波在不同拓扑下的距离律不同，自卫式和站外干扰不能共用同一套近似。

### 4.3 忽视与探测链的耦合

电子战层通常不会单独使用，而是需要与 `radar_equation`、`clutter` 和检测概率计算一起串联。
