# RCS 与电磁散射

> 本文对应 `include/xsf_math/core/rcs.hpp`。

## 1. 当前覆盖范围

RCS 模块当前支持：

- 常量 RCS
- 一维方位表
- 二维方位-俯仰表
- 频率相关 RCS
- 双基地角平分线近似
- Rayleigh / Mie / Optical 区间分类

## 2. 数据表达

RCS 值统一通过 `rcs_value` 表达，支持：

- `m2`
- `dbsm`

并提供：

- `from_m2(...)`
- `from_dbsm(...)`

这让上层模块可以按工程习惯在面积域和 dB 域之间切换。

## 3. 表驱动模型

应用已有实验或仿真数据时，可优先使用：

- `rcs_table_1d`
- `rcs_table_2d`
- `rcs_freq_dependent`

原因是这类模型更容易和目标库、签名库对接，而不是把 RCS 简化为固定常数。

## 4. 双基地近似

当前提供的双基地近似使用角平分线思想：

$$
\sigma_{bi} \approx \sigma_{mono}(\theta_{bisector})
$$

它适合做快速估算，但不应理解为严格的双基地散射求解器。

## 5. 与雷达链的关系

RCS 是 `radar_equation.hpp` 中的核心输入之一。

目标是否可探测，往往不是只由雷达功率决定，还强烈受：

- 目标朝向
- 工作频率
- 散射区间

影响。
