# 轨道力学

> 本文对应 `include/xsf_math/orbital/` 下的三个头文件：`kepler.hpp`、`j2.hpp`、`maneuvers.hpp`。

## 1. 当前实现能力

当前轨道模块分三层：

**开普勒主项** (`orbital/kepler.hpp`)

- 地球引力常数 `mu_earth`
- 经典轨道根数 `orbital_elements`
- 开普勒方程求解
- 真近点角/偏近点角/平近点角转换
- 二体开普勒传播 `propagate_kepler`
- 轨道根数到 ECI 状态转换 `elements_to_state`
- 卫星间视线可见性判断 `los_visible`

**J2 摄动** (`orbital/j2.hpp`)

- J2 常数 `j2_earth` 和参数 `j2_params`
- 惯性系 J2 加速度 `j2_acceleration`
- 长期漂移率 `compute_j2_secular_rates`（升交点赤经、近地点幅角、平近点角）
- 含 J2 岁差的传播 `propagate_j2_secular`

**轨道机动** (`orbital/maneuvers.hpp`)

- Hohmann 双脉冲转移 `hohmann_transfer`
- 圆化机动 `circularize_delta_v`
- 平面变换速度增量 `plane_change_delta_v` / `raan_inclination_change_delta_v`
- 倾角变换 `inclination_change_delta_v`

## 2. 轨道根数

`orbital_elements` 当前包含：

- 半长轴
- 偏心率
- 倾角
- 升交点赤经
- 近地点幅角
- 真近点角

并能直接导出：

- 周期
- 平均角速度
- 半径
- 近地点/远地点半径
- 对应速度

## 3. 开普勒传播

传播逻辑采用典型二体模型：

```mermaid
flowchart LR
    NU0["🌟 真近点角 ν₀"] --> E0["📐 偏近点角 E₀"]
    E0 --> M0["⏱️ 平近点角 M₀"]
    M0 --> MT["🔄 M(t) = M₀ + n·Δt"]
    MT --> ET["🔍 求解 E(t)<br/>开普勒方程迭代"]
    ET --> NUT["🌟 真近点角 ν(t)"]
    NUT --> STATE["📍 新 ECI 状态<br/>(r, v)"]

    style MT fill:#ffedd5
    style STATE fill:#e1f5e1
```

1. 从当前真近点角换算到偏近点角
2. 再换算到平近点角
3. 根据时间推进平近点角
4. 求解新的偏近点角
5. 反算新的真近点角

这适合快速轨道传播和示例验证。

## 4. 状态转换与可见性

当前库还提供：

- `elements_to_state(...)`
  将轨道根数转换为 ECI 中的位置和速度
- `los_visible(...)`
  判断两点连线是否被地球遮挡

这些能力足以支撑卫星间几何分析和简单可见性判定。

## 5. J2 摄动

`propagate_j2_secular` 在二体主项之外叠加了 J2 一阶长期漂移：

- 升交点赤经 $\dot{\Omega}$
- 近地点幅角 $\dot{\omega}$
- 平近点角 $\dot{M}$（含二体主项 $n$ 和 J2 修正）

对于工程轨道分析，这是解释"太阳同步、冻结轨道"现象的最小摄动集。

如果需要瞬时加速度级别的 J2 效应（例如与数值积分器耦合），使用 `j2_acceleration`，它直接返回 ECI 系下的加速度向量。

## 6. 轨道机动

`maneuvers.hpp` 把常见工程问题直接映射到速度增量估算：

| 问题 | 入口 |
|------|------|
| 共面圆轨道提升/降低 | `hohmann_transfer` 返回 $\Delta v_1$、$\Delta v_2$ 和转移半程时间 |
| 圆化当前轨道 | `circularize_delta_v` 返回当前速度与圆轨道速度差 |
| 纯倾角变换 | `inclination_change_delta_v` |
| 倾角 + 升交点联合变换 | `raan_inclination_change_delta_v`（先算两个轨道面法向的夹角，再算合成 $\Delta v$）|

## 7. 适用边界

当前实现偏向教育/工程估算，不包含：

- 高阶摄动（J4 以上）
- 大气阻力高保真模型
- 三体问题
- 地月日复杂摄动
- 太阳辐射压、固体潮等小量

因此更适合：

- 初步轨道分析
- 示例演示
- 需要轻量轨道几何的上层应用
