# 基础知识整理

本文件对应轨道动力学业务域，目的是在阅读 `orbital` 代码前，先把开普勒传播、摄动和轨道机动这三类问题分开理解。

## 1. 这一层解决什么问题

轨道动力学层关心的是：

- 如何用轨道根数表达状态
- 如何在二体问题下进行传播
- 长期摄动怎样改变轨道平面和近地点
- 常见轨道转移需要多少速度增量

因此它与近地飞行控制不同，更强调长期传播和轨道几何。

## 2. 阅读代码前应先掌握的概念

### 2.1 轨道根数

最常用的六根数包括：

- 半长轴
- 偏心率
- 轨道倾角
- 升交点赤经
- 近地点幅角
- 真近点角

代码里很多传播逻辑都是围绕这些量展开，而不是直接围绕位置速度状态量。

### 2.2 二体问题和 Kepler 传播

在最基础的近似下，轨道运动由中心引力决定。Kepler 传播解决的是这类理想二体问题。

### 2.3 J2 摄动

地球不是完美球体，赤道隆起会导致：

- 升交点赤经漂移
- 近地点幅角漂移

对长期轨道分析来说，这些项不能忽略。

### 2.4 轨道机动

从一个轨道到另一个轨道，最常见的问题是：

- 两个圆轨道之间如何转移
- 需要多少速度增量
- 时间代价有多大

## 3. 与代码的对应关系

- `include/xsf_math/orbital/kepler.hpp`
  提供二体轨道根数和基础传播。
- `include/xsf_math/orbital/j2.hpp`
  提供 J2 摄动和相关长期变化率。
- `include/xsf_math/orbital/maneuvers.hpp`
  提供常见轨道转移和速度增量估计。

## 4. 常见误区

### 4.1 把短期传播和长期传播混为一谈

短期传播里二体模型可能足够，但长期传播时 J2、阻力和更高阶扰动会累积出明显偏差。

### 4.2 只看位置，不看轨道几何

轨道问题很多时候更适合在根数空间分析，而不是一直停留在位置速度坐标。

### 4.3 把 Hohmann 转移当成所有机动的默认最优解

Hohmann 转移是经典低推力外的两脉冲最优解之一，但并不覆盖平面变换、低推力转移和复杂约束场景。
