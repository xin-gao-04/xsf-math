# 雷达信号处理基础

> 本文面向 `radar/` 模块，概述当前库中已经具备的雷达信号链路要素。

## 1. 当前覆盖范围

`xsf-math` 中的雷达相关能力主要包括：

- 发射机/接收机参数建模
- 单基地与双基地雷达方程
- 天线方向图
- 自由空间与大气传播损耗
- 杂波与 MTI 改善
- 干扰背景下的 SNR 评估
- 探测概率与作用距离估算

## 2. 雷达方程

单基地接收功率的核心形式为：

```text
Pr = (Pt * Gt * Gr * lambda^2 * sigma) / ((4*pi)^3 * R^4 * L)
```

当前实现见：

- `include/xsf_math/radar/radar_equation.hpp`

双基地情形则将 `R^4` 换成 `Rt^2 * Rr^2`。

## 3. 噪声、杂波与干扰

本库显式区分：

- `N`: 热噪声
- `C`: 杂波
- `J`: 干扰

因此可表达：

```text
SNR = S / (C + J + N)
```

对应组合结构：

- `snr_with_interference`

这让应用层可以把纯净雷达链与受干扰场景区分开来。

## 4. 传播模型

当前传播侧提供：

- `free_space_loss_db(...)`
- `blake_attenuation`
- `rain_attenuation`
- `lens_loss_db(...)`
- `multipath_model`
- `total_propagation_loss_db(...)`

这些能力适合做工程级估算，而不是高保真电磁场仿真。

## 5. 天线与方向图

当前方向图模型包括：

- 各向同性天线
- 余弦幂方向图
- sinc 平方方向图
- 查表方向图

对应头文件：

- `include/xsf_math/radar/antenna.hpp`

这足够支撑大多数作用距离、视场和波束偏置分析。

## 6. 杂波

杂波模块提供：

- 地表类型到典型 `gamma` 的映射
- 杂波斑块面积估算
- 杂波功率估算
- MTI 改善因子

对应头文件：

- `include/xsf_math/radar/clutter.hpp`

它适合下视雷达和基础信杂比评估。

## 7. 与测试/示例的对应关系

- `examples/radar_detection_example.cpp`
- `tests/test_radar.cpp`

这两个入口基本覆盖了当前雷达链的主要使用方式。
