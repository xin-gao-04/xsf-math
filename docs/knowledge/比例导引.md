# 比例导引与制导计算

> 本文对应 `include/xsf_math/guidance/proportional_nav.hpp`。

## 1. 当前实现能力

当前制导模块包括：

- `engagement_geometry`
- `proportional_nav`
- `augmented_proportional_nav`
- `pursuit_guidance`
- `accel_limiter`
- 阶段切换条件 `phase_transition`

## 2. 相对几何

`engagement_geometry` 统一封装了：

- 弹目相对位置
- 相对速度
- 斜距
- 闭合速度
- LOS 单位向量
- LOS 角速度
- 预计拦截时间

这使得不同制导律共享同一套几何输入。

## 3. 比例导引

当前 PN 形式为：

```text
a_cmd = N * (omega x V_weapon)
```

其中：

- `N` 是导航比
- `omega` 是 LOS 角速度
- `V_weapon` 是武器速度

## 4. 增强比例导引

APN 在 PN 的基础上额外叠加目标加速度估计：

```text
a_cmd = N * (omega x V_weapon) + (N/2) * a_target
```

这适合目标存在机动时的补偿。

## 5. 追踪导引

`pursuit_guidance` 采用更直观的几何方式：

- 让速度方向朝向目标当前位置
- 通过双叉乘构造机动平面的侧向加速度方向

这种方式实现简单，但通常不如 PN 节能。

## 6. 过载限制

`accel_limiter` 提供：

- 固定最大过载限幅
- 基于动压、面积、`CLmax`、质量的最大可用过载估算

这使制导计算能和气动能力自然衔接。

## 7. 阶段切换

当前支持的切换变量包括：

- 飞行时间
- 高度
- 速度 / 马赫数
- 动压
- 目标距离
- 地面距离
- 预计拦截时间
- LOS 方位/俯仰

这适合构建多阶段制导状态机。
