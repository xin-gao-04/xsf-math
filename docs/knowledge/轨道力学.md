# 轨道力学

> 本文对应 `include/xsf_math/orbital/kepler.hpp`。

## 1. 当前实现能力

当前轨道模块提供：

- 地球引力常数 `mu_earth`
- 经典轨道根数 `orbital_elements`
- 开普勒方程求解
- 真近点角/偏近点角/平近点角转换
- 二体开普勒传播
- 轨道根数到 ECI 状态转换
- 卫星间视线可见性判断
- 简化轨道阻力减速度估算

## 2. 轨道根数

`orbital_elements` 当前包含：

- 半长轴
- 偏心率
- 倾角
- 升交点赤经
- 近地点幅角
- 真近点角

并能直接导出：

- 周期
- 平均角速度
- 半径
- 近地点/远地点半径
- 对应速度

## 3. 开普勒传播

传播逻辑采用典型二体模型：

1. 从当前真近点角换算到偏近点角
2. 再换算到平近点角
3. 根据时间推进平近点角
4. 求解新的偏近点角
5. 反算新的真近点角

这适合快速轨道传播和示例验证。

## 4. 状态转换与可见性

当前库还提供：

- `elements_to_state(...)`
  将轨道根数转换为 ECI 中的位置和速度
- `los_visible(...)`
  判断两点连线是否被地球遮挡

这些能力足以支撑卫星间几何分析和简单可见性判定。

## 5. 适用边界

当前实现偏向教育/工程估算，不包含：

- 高阶摄动
- J2/J4 摄动
- 大气阻力高保真模型
- 三体问题
- 地月日复杂摄动

因此更适合：

- 初步轨道分析
- 示例演示
- 需要轻量轨道几何的上层应用
